📚 Panduan Lengkap SAPRAS Digital

🚀 Memulai Aplikasi

  • • Aplikasi siap digunakan tanpa instalasi
  • • Data tersimpan otomatis di browser
  • • Mulai dengan menambah barang pertama
  • • Kode barang dibuat otomatis (SPR0001, SPR0002, dst)

📝 Menambah Inventaris

  • • Nama Barang: Gunakan nama yang spesifik
  • • Kategori: Pilih sesuai jenis barang
  • • Lokasi: Tentukan ruang penyimpanan
  • • Kondisi: Nilai secara objektif
  • • Sumber Dana: Catat asal pembiayaan

🔍 Pencarian & Filter

  • • Pencarian: Ketik nama atau kode barang
  • • Filter Kategori: Tampilkan per jenis
  • • Filter Lokasi: Cari per ruangan
  • • Filter Kondisi: Sortir berdasarkan kondisi
  • • Kombinasikan filter untuk hasil spesifik

⚡ Fitur Lanjutan

  • • Barcode: Generate untuk labeling
  • • Edit: Klik tombol edit untuk mengubah
  • • Hapus: Konfirmasi sebelum menghapus
  • • Statistik: Pantau total dan nilai aset
  • • Stok Rendah: Alert untuk barang < 5 unit

🎯 Tips Akurasi Data

  • • Gunakan nama lengkap dan merk/brand
  • • Pilih kategori yang paling sesuai
  • • Tentukan lokasi dengan tepat
  • • Nilai kondisi secara objektif
  • • Isi keterangan untuk detail tambahan

⚠️ Hal Penting

  • • Data tersimpan di browser lokal
  • • Jangan clear data/cache browser
  • • Backup data secara berkala
  • • Gunakan browser yang sama
  • • Akses dari komputer yang sama

📋 Kode Kategori & Lokasi

Kategori:

  • • PPU - Peralatan Praktek Umum
  • • PPK - Peralatan Praktek Kejuruan
  • • AKB - Alat Kebersihan
  • • AKK - Alat Keamanan & K3
  • • ABI - Aset Bangunan Infrastruktur

Lokasi Utama:

  • • RKS - Ruang Kepala Sekolah
  • • RGR - Ruang Guru
  • • RTU - Ruang Tata Usaha
  • • KLS-X/XI/XII - Kelas per tingkat
  • • LAB - Laboratorium
